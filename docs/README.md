# Integration Docs

Start here if you want the fastest path to understanding how the backend is wired and how a frontend should consume it.

## Guides

- [Backend Integration Guide](./backend-integration-guide.md)
- [Frontend WebSocket and Redis Integration Guide](./frontend-websocket-redis-integration.md)

## What Each Guide Covers

- Backend guide: architecture, auth, REST, websocket, Redis, storage, security, and operational notes.
- Frontend guide: connection setup, subscriptions, publish actions, payloads, and implementation checklist.

## Practical Order

1. Read the backend guide to understand the system contract.
2. Read the frontend guide to wire the live client.
3. Treat REST as the source of truth and websocket as the live update layer.
